## 導入方法
1. PHP と Composer をセットアップ
2. Composer で Laravel をインストール
```sh
composer create-project laravel/laravel laravel-blocs
cd laravel-blocs
```

3. Composer で BLOCS をインストール
```sh
laravel-blocs % composer require blocs/blocs    
```
